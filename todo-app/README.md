# Todo-app



## 개요

해당 페이지는 React의 useState를 통해 상태를 관리하고 상태에 따른 목록을 보여주는 프로젝트 입니다.

![GIF](https://user-images.githubusercontent.com/29428714/161429578-3092f8af-5b42-4306-b9c8-e83383005d47.gif)


## 기능

- Input Box에 Text 입력 가능
- Enter를 통해 Submit 가능
- 제출 시 목록에 추가
